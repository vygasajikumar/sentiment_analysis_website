😊 Sentiment Analysis Web Application

A simple yet interactive web application that analyzes user-entered text and classifies it as Positive, Negative, or Neutral. Built using HTML, CSS, and JavaScript, this project demonstrates the fundamentals of sentiment analysis and front-end web development.

---

📖 Overview

Understanding the sentiment behind user reviews and feedback is an important aspect of modern applications. This project provides a basic implementation of sentiment analysis by evaluating text against predefined positive and negative keywords and determining the overall sentiment.

The application offers an easy-to-use interface where users can enter their opinions, comments, or reviews and instantly receive sentiment feedback.

---

✨ Features

- 📝 User-friendly text input interface
- ⚡ Real-time sentiment evaluation
- 😊 Positive sentiment detection
- 😐 Neutral sentiment detection
- 😞 Negative sentiment detection
- 🚀 Fast processing without external APIs
- 📱 Lightweight and responsive design
- 🎯 Beginner-friendly implementation

---

🔍 How It Works

1. The user enters a review or comment.
2. The application scans the text for predefined positive and negative keywords.
3. A sentiment score is calculated based on the detected words.
4. The application classifies the sentiment as:
   - 😊 Positive
   - 😐 Neutral
   - 😞 Negative
5. The result is displayed instantly.
